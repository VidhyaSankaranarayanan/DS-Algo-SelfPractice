**KANDANE ALGORITHM**

This technique is useful for finding subarrays inside an array or substring in a string

It is a subset of dynamic programming

Time complexity: O(n) & Space Complexity: O(1)

Helpful when the array has mix of positive & negative numbers

**Approach** 

1. Add the subsequent elements
2. If the sum is greater than zero, continue to add window sum
3. If the sum is lesser than zero, reset the window sum as zero

**BINARY SEARCH**

This algorithm is efficient in terms of time complexity which is O(logn)

It is based on Divide & Conquer technique

Condition: Test data should be sorted

**Approach**

1. 2 pointers - low(first element) & high(last element)
2. Find mid between low & high
3. If the mid matches the condition, return
4. If mid value is lesser than expected value, increment low to mid + 1
5. If mid value is greater than expected value, decrement high to mid - 1

**RECURSION**

Same function will be called again and again until the breakpoint is reached.
Lines of code will be less

**5 STEPS**

1. Write base case/ breakpoint for simplest possible input
2. Positive test scenario
3. Edge & Negative test scenario
4. Generalize the pattern - formula/logic to arrive the pattern
5. Code using the base case & formula/logic
